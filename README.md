# Jupyter Notebooks

Collection of ML experiments, tutorials, and analysis notebooks.

## Notebooks
- **01_data_exploration**: Pandas profiling and visualization
- **02_feature_engineering**: Feature creation and selection
- **03_model_comparison**: Comparing sklearn models
- **04_deep_learning**: PyTorch basics
- **05_nlp_basics**: Text classification

## Setup
```bash
pip install -r requirements.txt
jupyter notebook
```
